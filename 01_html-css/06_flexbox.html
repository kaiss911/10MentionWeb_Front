C-direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction ,on peut les positionner verticalement ou encore les inverser. il peut prendre les valeurs suivantes:
->row : organiser sur une ligne (par defaut)
->column : organiser sur une colonne
->row-reverse : organiser sur une ligne mais en ordre inversé
->column-reverse : organiser sur une olonne mais en ordre inversé
