le CSS : Cascading Style Sheet

Le CSS permet de faire du style sur nos page html

Le CSS dans la balise

La premiere façon de faire du style dans un element est de declarer l'attribut style sur celui-là. Ce n'est pas la meilleur de faire

Je suis un paragraphe stylé !

Le CSS dans une feuille de style externe

La dernière est la meilleure façon pour faire du style : on crée un dossier css avec un fichier "style.css" à l'interieur il faudra alors lier nos deux fichier gràce a la balise link. on pourra allors faire du style sur les elements qui se trouve sur notre page. Exemple, on declare que tout les paragraphes doivent etre en bleu

Le CSS dans l'en tête

Une seconde façon de faire du style sur notre page et de mettre la balise style dans la partie head. Cette méthode n'est toujours pas la plus conseillée mais elle permet plus de choses que la méthode balise en ligne, on peut désormais grâce a cette méthode viser plusieurs éléments d'un coup. Exemple, on mets du gris en fond sur le corps de la page, les h2 en rouge et les h3 avec un fond rose
